1️⃣ api.js (≈ 250 سطر)
   
   SwmiAPI Class:
   ├─ loadData()          → تحميل data.json
   ├─ getData()           → الحصول على كل البيانات
   ├─ getFastingRules()   → أحكام الصيام
   ├─ getHadiths()        → الأحاديث الصحيحة
   ├─ getCities()         → المدن التونسية
   ├─ getPrayerTimes()    → أوقات الصلاة
   ├─ getAdhkar()         → الأذكار والأدعية
   ├─ getQuranVerses()    → آيات القرآن
   ├─ getFAQs()           → الأسئلة الشائعة
   └─ getHealthTips()     → النصائح الصحية

   SwmiStorage Class:
   ├─ saveUser()          → حفظ بيانات المستخدم
   ├─ getUser()           → الحصول على المستخدم
   ├─ addFastingDay()     → إضافة يوم صيام
   ├─ getFastingDays()    → الحصول على أيام الصيام
   ├─ addFavoriteDua()    → حفظ دعاء مفضل
   ├─ getFavoriteDuas()   → الأدعية المفضلة
   ├─ saveSettings()      → حفظ الإعدادات
   ├─ getSettings()       → الإعدادات
   ├─ getTasbeehCount()   → عداد التسبيح
   ├─ exportData()        → تصدير كل البيانات
   ├─ importData()        → استيراد البيانات
   └─ clearAllData()      → حذف كل البيانات

   SwmiAuth Class:
   ├─ register()          → تسجيل مستخدم جديد
   ├─ login()             → تسجيل الدخول
   ├─ logout()            → تسجيل الخروج
   ├─ getCurrentUser()    → المستخدم الحالي
   ├─ isLoggedIn()        → تحقق من الدخول
   ├─ updateProfile()     → تحديث الملف الشخصي
   ├─ changePassword()    → تغيير كلمة المرور
   ├─ deleteAccount()     → حذف الحساب
   ├─ getPublicProfile()  → ملف عام للمستخدم
   └─ generateAvatar()    → توليد صورة مستخدم

   SwmiProgress Class:
   ├─ recordFastingDay()  → تسجيل يوم صيام
   ├─ getFastingHistory() → تاريخ الصيام
   ├─ getMonthProgress()  → تقدم الشهر
   ├─ getCurrentStreak()  → السلسلة الحالية
   ├─ getOverallStats()   → إحصائيات عامة
   ├─ getMonthCalendar()  → تقويم الشهر
   ├─ isFastedToday()     → هل صام اليوم
   └─ getNextMilestone()  → الهدف القادم
