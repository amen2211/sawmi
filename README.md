## 🎯 الميزات الرئيسية

### 1. نظام المصادقة (Authentication)
```javascript
// التسجيل
auth.register({
    name: 'محمد',
    email: 'user@example.com',
    city: 'tunis'
});

// تسجيل الدخول
auth.login('user@example.com', 'password');

// التحقق من الدخول
if (auth.isLoggedIn()) {
    // مستخدم مسجل دخول
}
```

### 2. إدارة البيانات (Storage)
```javascript
// حفظ يوم صيام
storage.addFastingDay('2026-03-02', 'completed');

// الحصول على الإحصائيات
const stats = storage.getStats();
// { username, totalFastingDays, completedDays, streak }

// حفظ الأدعية المفضلة
storage.addFavoriteDua(duaObject);

// الحصول على المفضلة
const duas = storage.getFavoriteDuas();
```

### 3. تتبع التقدم (Progress Tracking)
```javascript
// تسجيل يوم صيام
progress.recordFastingDay('2026-03-02', 'completed');

// الحصول على السلسلة الحالية
const streak = progress.getCurrentStreak();

// الحصول على ملخص الشهر
const monthProgress = progress.getMonthProgress();

// الحصول على الهدف التالي
const milestone = progress.getNextMilestone();
// { target: 30, current: 15, remaining: 15, percentage: 50 }
```

### 4. API البيانات (Data API)
```javascript
// جلب أحكام الصيام
const rules = await api.getFastingRules();

// جلب الحاديث
const hadiths = await api.getAllHadiths();

// جلب الأدعية
const adhkar = await api.getAllAdhkar();

// جلب الآيات القرآنية
const verses = await api.getQuranVerses();

// جلب أوقات الصلاة لمدينة معينة (باستخدام API Aladhan مع method=3، timezone Africa/Tunis؛ لا توجد أوقات ثابتة)
// سيعرض الموقع رسالة خطأ إذا تعذّر الحصول على البيانات.
const times = await api.getPrayerTimes('tunis');
```

## 🎓 شرح البنية التقنية

### المعمارية
```
User Browser
    ├── HTML (Structure)
    ├── CSS (Styling)
    ├── JavaScript (Behavior)
    │   ├── script.js (Main)
    │   ├── api.js (Data Management)
    │   └── auth.js (Authentication)
    └── LocalStorage (Data Persistence)
        └── data.json (Static Data)
```

### تسلسل البيانات
```
data.json → api.js → (cache in memory)
         ↓
localStorage → storage.js
         ↓
User Interface (HTML/CSS)
         ↓
User Interaction (script.js)
```
